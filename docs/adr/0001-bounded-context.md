# ADR-0001: Bounded Context Для Attribution Service

## Статус
Принято

## Дата
2026-04-05

## Контекст
Логика атрибуции и скидок выделена из `course_service`, чтобы держать channel/promo concerns независимыми.

## Решение
`attribution-service` владеет lifecycle токенов, скидочными политиками и отчетностью по конверсиям.
`course_service` потребляет решения/факты атрибуции через API.

## Последствия
- более чистые доменные границы
- независимое масштабирование attribution/reporting
- явные интеграционные контракты
